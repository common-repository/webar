== Description ==
This plugin will help user to display the Apple AR (usdz) content to any WordPress Website. You can assign placeholder image and specify image dimensions. It also allows to upload the USDZ file to media section.


== Installation ==

### INSTALL WebAR FROM WITHIN WORDPRESS

1. Visit the plugins page within your dashboard and select ‘Add New’;
2. Search for ‘WebAR’;
3. Activate WebAR from your Plugins page;

### INSTALL WebAR MANUALLY

1. Upload the ‘webar’ folder to the /wp-content/plugins/ directory;
2. Activate the WebAR plugin through the ‘Plugins’ menu in WordPress;
